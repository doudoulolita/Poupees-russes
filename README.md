Jeu basique en cours pour tester Godot mono en C#.

Modèles 3D réalisés avec Blender.

Textures poupées russes faites avec Gimp.
Texture bois CCO : https://polyhaven.com/a/dark_wood

Idée proposée lors du Blender User Group de Paris.

Le BUG de Paris se réunit tous les 1ers samedis du mois au Carrefour Numérique de la Cité des Sciences.

# Comment jouer ?

Importer le jeu dans Godot-mono pour l'éditer ou y jouer.

La poupée bleue se dépace avec les flèches et saute avec la touche Espace.

Elle perd des points quand une fleur la touche.

Elle gagne des points quand elle touche une poupée rose.
